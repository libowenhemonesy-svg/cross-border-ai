# Amazon AI Product Analyzer

Amazon 商品页 AI 选品分析插件 - 提取商品信息并展示分析结果

## 功能介绍

这是一个 Chrome 浏览器插件，用于从 Amazon 商品详情页提取商品信息并展示在插件弹窗中。

### 主要功能

1. **商品信息提取**：自动从 Amazon 商品页面提取以下信息：
   - 商品标题
   - 商品价格
   - 商品评分
   - 评论数量
   - 五点描述
   - ASIN 编码
   - 当前页面 URL

2. **弹窗展示**：以卡片式布局美观地展示提取到的商品信息

3. **错误处理**：如果页面字段提取失败，会显示"未找到"而不是报错

### 技术特点

- 使用 Chrome Extension Manifest V3
- 原生 HTML、CSS、JavaScript 开发
- 代码结构清晰，注释完整，适合新手学习
- 只读取当前页面可见信息，不做批量爬虫

## 安装方法

### 步骤 1：下载插件代码

将整个 `amazon-ai-product-analyzer` 文件夹下载到本地电脑。

### 步骤 2：打开 Chrome 扩展管理页面

1. 打开 Chrome 浏览器
2. 在地址栏输入 `chrome://extensions/` 并回车
3. 或者点击浏览器右上角的三个点 → 更多工具 → 扩展程序

### 步骤 3：开启开发者模式

在扩展管理页面右上角，找到"开发者模式"开关并打开。

### 步骤 4：加载插件

1. 点击左上角的"加载已解压的扩展程序"按钮
2. 在弹出的文件选择窗口中，选择 `amazon-ai-product-analyzer` 文件夹
3. 点击"选择文件夹"

### 步骤 5：确认安装

安装成功后，你会在扩展列表中看到"Amazon AI Product Analyzer"插件。

## 使用方法

### 步骤 1：打开 Amazon 商品页面

1. 打开 Chrome 浏览器
2. 访问 https://www.amazon.com
3. 搜索并打开任意商品详情页

### 步骤 2：点击插件图标

在浏览器右上角的工具栏中，找到插件图标（购物车+闪电图标）并点击。

### 步骤 3：提取商品信息

在弹出的插件窗口中，点击"提取当前商品信息"按钮。

### 步骤 4：查看结果

插件会自动提取当前页面的商品信息并展示在弹窗中。

## 测试方法

### 测试场景 1：正常商品页面

1. 打开一个 Amazon 商品详情页（例如：https://www.amazon.com/dp/B09V3KXJPB）
2. 点击插件图标
3. 点击"提取当前商品信息"
4. 验证是否正确提取了商品标题、价格、评分等信息

### 测试场景 2：非 Amazon 页面

1. 打开任意非 Amazon 网站（例如：https://www.google.com）
2. 点击插件图标
3. 验证是否显示"请打开 Amazon 商品详情页后再使用此插件"提示

### 测试场景 3：Amazon 首页（非商品页）

1. 打开 Amazon 首页（https://www.amazon.com）
2. 点击插件图标
3. 点击"提取当前商品信息"
4. 验证是否能正确处理（大部分字段会显示"未找到"）

## 当前版本限制

1. **只支持 amazon.com**：目前只支持美国站 Amazon，后续会扩展其他站点

2. **页面结构依赖**：插件依赖 Amazon 页面的 HTML 结构，如果 Amazon 更新页面设计，部分字段可能提取失败

3. **不支持动态加载**：如果商品信息是通过 JavaScript 动态加载的，可能无法提取

4. **无 AI 分析功能**：第一版只做信息提取和展示，不包含 AI 分析功能

5. **单页面限制**：只能提取当前打开的单个商品页面信息，不支持批量处理

## 常见问题

### Q: 为什么有些字段显示"未找到"？

A: 可能有以下原因：
1. 该商品页面没有这个信息
2. Amazon 更新了页面结构
3. 信息是动态加载的，插件无法获取

### Q: 插件会收集我的个人信息吗？

A: 不会。插件只读取当前页面的商品信息，不会收集或上传任何个人信息。

### Q: 可以在其他浏览器使用吗？

A: 目前只支持 Chrome 浏览器。后续可能会支持 Edge、Firefox 等浏览器。

### Q: 插件会影响网页加载速度吗？

A: 不会。插件只在用户点击按钮时才会运行，不会自动运行或影响网页性能。

## 开发说明

### 项目结构

```
amazon-ai-product-analyzer/
├── manifest.json      # 插件配置文件
├── popup.html         # 弹窗页面
├── popup.css          # 弹窗样式
├── popup.js           # 弹窗脚本
├── content.js         # 内容脚本
├── icons/
│   └── icon.svg       # 插件图标
└── README.md          # 说明文档
```

### 核心文件说明

1. **manifest.json**：插件的配置文件，定义了插件的权限、入口文件等
2. **content.js**：注入到 Amazon 页面的脚本，负责提取商品信息
3. **popup.html/css/js**：插件弹窗的界面和逻辑
4. **icon.svg**：插件图标
