# 6 天冲刺复习计划（5/20 → 5/26）

## 总体策略

- **每天 4-6 小时专注学习**，上午 2-3 小时 + 下午 2-3 小时
- **20% 理论 + 80% 实践**：每学一个概念就写代码验证
- **晚上 30 分钟回顾**：用费曼学习法自问自答
- **准备一个 GitHub 项目**：面试时展示代码

## 高频自测题目

每天随机抽 3 题自答：

1. async/await 的原理是什么？事件循环怎么工作的？
2. RAG 的完整流程？每个步骤的技术选型？
3. 向量相似度有哪几种度量方式？有什么区别？
4. 怎么控制 LLM API 的并发？Semaphore 怎么用？
5. Prompt Injection 是什么？怎么防护？
6. LangChain 有哪些优点和缺点？
7. 什么是 Function Calling？完整流程是怎样的？
8. 流式输出（SSE）前后端怎么实现？
9. LLM 缓存怎么做？精确匹配 vs 语义缓存？
10. ReAct Agent 的循环是怎样的？
11. 文档分块策略有哪些？chunk_overlap 有什么用？
12. Token 怎么计算？怎么估算一次调用的成本？
13. MCP 协议是什么？解决了什么问题？
14. LLM 请求失败了怎么处理？（重试、降级、兜底）
15. K8s 的 HPA 是什么？AI 服务怎么配？
16. 怎么评估 RAG 系统的效果？
17. 多 Agent 协作有哪些常见模式？
18. Embedding 模型怎么选？维度和精度/成本的关系？
19. 怎么设计一个智能客服的转人工逻辑？
20. 你最近关注了 AI 领域的哪些新进展？
