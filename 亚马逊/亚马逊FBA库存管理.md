# 亚马逊FBA库存管理

> 断货 = 权重清零，滞销 = 利润被仓储费吃掉。库存管理是亚马逊卖家最容易忽视、却最致命的环节。

---

## 一、FBA 基础认知

### 1.1 FBA 全流程

```
供应商发货 → 头程物流 → 亚马逊仓库 → 上架可售 → 顾客下单 → 亚马逊配送 → 退货处理
     ↑                                          ↓
     └──────── 补货（一切的核心） ────────────────┘
```

### 1.2 FBA 费用结构

| 费用类型 | 计算方式 | 备注 |
|----------|----------|------|
| **月度仓储费** | 按立方英尺 × 月费率 | 1~9月低，10~12月涨 3 倍+ |
| **长期仓储费** | 存放超过 365 天 | 极高，必须避免 |
| **配送费** | 按尺寸分段 + 重量 | 下单时扣 |
| **移除/弃置费** | 按件收费 | 清理滞销库存时产生 |
| **入库配置费** | 按件收费 | 2024 年起新增 |

> ⚠️ **旺季仓储费暴增**：10~12 月的仓储费约为平时的 3 倍。如果在 9 月底有大量库存，10 月的账单会让人怀疑人生。

## 二、核心库存指标

### 2.1 IPI（库存绩效指标）

```
IPI 是亚马逊给你的库存管理打分（0~1000）
IPI < 400 → 仓储容量受限 → 无法创建新发货计划
IPI > 550 → 仓储容量无限制
```

**IPI 提升方法**：
- 提高售出率
- 减少冗余库存
- 修复无在售信息的产品

### 2.2 售罄率（Sell-Through Rate）

```
售罄率 = 过去 90 天售出数量 ÷ 同期平均可售库存 × 100%

健康值：> 90 天库存 = 售罄率约 33%/月
```

### 2.3 库存可售天数

```
可售天数 = 当前 FBA 库存 ÷ 日均销量

新品期：30~45 天
稳定期：45~60 天
旺季前：60~90 天
```

## 三、安全库存计算公式

### 3.1 基础公式

```
安全库存量 = 日均销量 × (采购周期 + 物流周期 + 亚马逊上架周期) × 安全系数

安全系数：
  新品（销量不确定）→ 1.5
  稳定品（销量稳定）→ 1.2
  老品（销量下滑中）→ 1.0
```

### 3.2 补货点计算

```
补货点 = 日均销量 × 补货前置期 + 安全库存

前置期 = 向供应商下单到货物 → 头程运输 → 亚马逊签收上架

例：日均 10 单，前置期 30 天，安全库存 100 件
  补货点 = 10 × 30 + 100 = 400 件
  → 当 FBA 库存降到 400 件时，必须触发补货
```

### 3.3 库存推演示例

```
假设：
  日均销量：15 单
  采购周期：15 天（下单到出厂）
  头程物流：18 天（海运美西）
  亚马逊上架：7 天（签收到可售）
  安全系数：1.3

计算：
  安全库存 = 15 × (15+18+7) × 1.3 = 15 × 40 × 1.3 = 780 件
  补货点 = 15 × 40 + 780 = 1,380 件
  
你的 FBA 后台库存降到 1,380 件的那一刻 → 立即下单补货
```

## 四、头程物流策略

### 4.1 物流方式对比

| 方式 | 时效 | 成本 | 适用 |
|------|------|------|------|
| **海运快船** | 15~20天（美西） | $1.5~$3/kg | 主力补货 |
| **海运普船** | 25~35天 | $1~$2/kg | 不急的大批量 |
| **空运** | 5~8天 | $15~$25/kg | 新品首发/紧急补货 |
| **快递（DHL/UPS）** | 3~5天 | $25~$35/kg | 样品/超紧急 |
| **中欧班列** | 18~25天 | 中等 | 欧洲站备选 |

### 4.2 物流组合策略

```
常规补货（70%）：海运快船 — 成本与时效的最优平衡
应急补货（20%）：空运     — 销量超预期的快速响应
试单/新品（10%）：空运     — 首批 200~500 件快速测款
```

### 4.3 分仓与合仓

```
分仓（亚马逊自动分到多个仓库）：
  ✓ 配送更快（就近发货）
  ✗ 头程复杂，小批次运费高
  
合仓（设置 Inventory Placement）：
  ✓ 头程简单
  ✗ 需要支付合仓费（$0.30/件起）
  
建议：单品 < 500 件用合仓，大批量接受分仓
```

## 五、库存健康度监控

### 5.1 库存分类管理

| 类型 | 定义 | 行动 |
|------|------|------|
| **爆款库存** | 售罄率 > 50%/月 | 优先补货，确保不断 |
| **正常库存** | 售罄率 20%~50%/月 | 正常补货节奏 |
| **慢动库存** | 售罄率 10%~20%/月 | 降价/广告/移除计划 |
| **死库存** | 售罄率 < 10%/月 | 清仓/弃置/Outlet |

### 5.2 库存预警机制

```
🟢 绿色（健康）：可售天数 30~60 天
🟡 黄色（关注）：可售天数 15~30 天或 > 90 天
🔴 红色（行动）：可售天数 < 15 天（即将断货）
                 或可售天数 > 120 天（即将产生长期仓储费）
```

### 5.3 库存周转管理表

建议用 Excel/Google Sheets 维护一份表格：

```
SKU | 当前FBA库存 | 在途库存 | 日均销量 | 可售天数 | 补货点 | 状态
SKU001 | 800 | 500 | 15 | 53 | 600 | 🟢
SKU002 | 200 | 0   | 10 | 20 | 400 | 🔴 需要补货！
SKU003 | 2000| 0   | 8  | 250| 300 | 🔴 滞销风险！
```

## 六、季节性库存管理

### 6.1 旺季备货公式

```
旺季备货量 = (旺季预估日均销量 × 旺季天数) + 正常安全库存

注意：
  - 10~12 月仓储费暴涨，不宜过早大量入仓
  - 入仓时间建议：
    Prime Day（7月）：5月底~6月初入仓
    黑五/圣诞：9月底~10月中旬入仓
```

### 6.2 淡季库存收缩

```
淡季到来前 1 个月：
  → 减少补货量到正常水平的 60%~70%
  → 停止非必要的促销折扣
  → 关注长期仓储费预警
```

## 七、滞销库存处理

### 7.1 处理优先级

```
售罄率 30%~50%  →  可以等，维持正常策略
售罄率 10%~30%  →  加大广告/优惠券，加速出清
售罄率 < 10%    →  激进处理：大折扣 Outlet / 移除 / 弃置
```

### 7.2 处理手段对比

| 手段 | 回款比例 | 速度 | 适合 |
|------|----------|------|------|
| **Outlet 促销** | 60%~80% | 中 | 有品牌价值的产品 |
| **大幅优惠券（30%+）** | 50%~70% | 快 | 有需求的慢动品 |
| **社交媒体促销代码** | 50%~60% | 快 | 清仓走量 |
| **移除到海外仓** | 0%（暂时） | 中 | 后面可自发货或换标 |
| **弃置** | 0%（还要付费）| 快 | 彻底不要了 |

> 💰 决策逻辑：如果仓储费 + 长期仓储费 > 弃置费，直接弃置更划算。

## 九、FBA 库存管理常见错误

| 错误 | 后果 | 正确做法 |
|------|------|----------|
| **第一批货就大量备货** | 产品不行全部滞销 | 首单 200~500 件测款 |
| **只看销量不看来单** | 销量稳定但断货了 | 每天看可售天数 |
| **旺季前疯狂备货** | 节后滞销+仓储费爆炸 | 淡化旺季=销售额+20%，非翻倍 |
| **忽略移除/弃置费** | 清仓反而亏更多 | 算一下清仓净利润，可能弃置更划算 |
| **海运太慢，空运太贵** | 两头不讨好 | 组合策略：70% 海运 + 30% 空运 |
| **不设补货点** | 完全凭感觉补货 | 用公式算出补货点，严格执行 |

## 十、库存管理 Checklist

### 每周
- [ ] 检查各 SKU 可售天数
- [ ] 确认在途库存的物流状态
- [ ] 标记进入 🟡 或 🔴 预警的产品

### 每月
- [ ] 计算售罄率，做库存分类
- [ ] 更新安全库存量和补货点
- [ ] 检查 IPI 分数变化

### 每季
- [ ] 审查滞销库存，制定清仓计划
- [ ] 准备旺季备货计划
- [ ] 与物流商确认头程时效和价格
