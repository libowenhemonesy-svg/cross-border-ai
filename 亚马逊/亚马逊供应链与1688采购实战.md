# 亚马逊供应链与1688采购实战

> 产品是根，供应链是命。前端运营做得再好，供应链掉链子，一切归零。

---

## 一、供应链的底层认知

### 1.1 供应链 vs 单纯采购

```
单纯采购思维：找到便宜的货 → 下单 → 收货 → 发走
供应链思维：    供应商筛选 → 品控体系 → 交货管理 → 成本优化 → 持续迭代
```

| 维度 | 采购思维 | 供应链思维 |
|------|----------|------------|
| 供应商关系 | 买卖关系 | 合作伙伴关系 |
| 关注点 | 单价 | 总成本（含质量、交期、沟通） |
| 时间跨度 | 单次 | 持续迭代 |
| 风险控制 | 无 | 备选供应商、质检流程 |

### 1.2 好供应商的五个标准

```
✅ 响应快 — 24 小时内回复，沟通顺畅
✅ 交期准 — 说 15 天就是 15 天，不含水分
✅ 品质稳 — 三次样品一致，大货和样品无差异
✅ 配合度高 — 愿意接受小单、定制、改进
✅ 价格合理 — 不是最便宜，而是在品质前提下最优
```

## 二、1688 选供应商全流程

### 2.1 搜索与筛选

```
搜索关键词策略：
  不要只搜产品名 → 搜「产品名 + 跨境/外贸/亚马逊」
  例：不锈钢水杯 → 「不锈钢水杯 亚马逊」「不锈钢水杯 外贸」

筛选条件：
  □ 经营模式：生产厂家（非经销/贸易）
  □ 诚信通年限：≥ 3 年
  □ 回头率：≥ 20%
  □ 响应率：≥ 95%
  □ 发货速度：高于行业平均
  □ 有实地认证 / 验厂报告
  □ 支持 OEM/ODM
```

### 2.2 初筛沟通话术模板

> **第一次询盘（一句话表明你是做亚马逊的）**：
>
> "你好，我是做亚马逊美国站的，对你们这款 XX 产品很感兴趣。请问这款支持 OEM 贴牌吗？MOQ（最小起订量）多少？有没有做过亚马逊的客户？麻烦发一份报价单和产品规格表，谢谢。"

**为什么要这样说**：
- 说「亚马逊美国站」→ 供应商知道你要求高，重视品控
- 问「有没有做过亚马逊的客户」→ 有经验=包装合规=知道FBA要求
- 问 MOQ → 判断对方是否接受小批量试单

### 2.3 索要样品

```
样品阶段三步骤：

第 1 步：索要图片/视频（免费）
  → 初步评估产品外观、做工

第 2 步：购买现有样品（付费，但值）
  → 拿 2~3 家样品对比
  → 同一家拿 2 个，看一致性

第 3 步：定制打样（如需要 OEM）
  → 确认 Logo 位置、包装方案
  → 拿到最终大货确认样
```

> 💰 样品费+运费一份约 ¥50~200，千万别省。一份样品能避免几万的库存踩坑。

## 三、验厂与品控

### 3.1 小卖家也能做的「轻验厂」

没有预算飞到工厂？做以下三步替代：

```
远程验厂三步走：

1. 营业执照查询
  → 天眼查/企查查：是否真实存在、有无经营异常、有无法律诉讼

2. 视频验厂
  → 让工厂视频连线，看生产线、原材料仓库、成品仓
  → 看工人状态、设备新旧、车间整洁度

3. 同行口碑
  → 1688 评价区、跨境社群打听、问工厂老客户（用天眼查看关联公司）
```

### 3.2 质检标准制定

```
包装前必须确认的质检清单：

□ 外观：无划痕、色差、变形——
□ 尺寸：长/宽/高/重量 误差在 ±3% 以内
□ 功能：正常使用无异常
□ 配件：数量齐全，无遗漏
□ Logo：位置正确、清晰、不脱落
□ 包装：彩盒/白盒无破损、印刷清晰
□ 条码：FNSKU 可扫描（用手机扫码 APP 验证）
□ 安全：无尖锐边角、无异味、无有害物质

AQL 抽检标准（一般使用 Level II）：
  500 件以下 → 全检
  500~3000 件 → 抽检 80~125 件
  3000 件以上 → 抽检 200~315 件
```

## 四、与供应商谈判的关键点

### 4.1 报价谈判

| 要点 | 怎么谈 |
|------|--------|
| **不要一上来就砍价** | 先建立信任，了解成本构成 |
| **用量换价** | "首批 500 件试单，如果品质稳定，后续每单 3000+" |
| **用长期合作换支持** | "我们是长期做的，能给个老客户价吗" |
| **用付款方式换价** | "如果价格再降 5%，我可以预付 50%" |
| **对比报价** | "另一家给 ¥X，你们能给到多少"（谨慎用，容易伤关系） |

### 4.2 付款方式

```
推荐节奏：

首批试单：50% 定金 + 50% 发货前付清
   ↓（合作 2~3 次后）
稳定合作：30% 定金 + 70% 发货前付清
   ↓（长期稳定后）
优质供应商：30% 定金 + 70% 收货后 15~30 天付清
```

> ⚠️ 永远不要 100% 预付。不管对方说得多么靠谱。

### 4.3 合同关键条款

```
必须写入合同：
  □ 产品规格、材质、颜色（附确认样照片）
  □ 单价和总金额（含税/不含税）
  □ 交期（明确年/月/日，写明逾期处罚）
  □ 质量标准和验收标准
  □ 包装要求（FBA 包装标准）
  □ 付款方式和节点
  □ 违约责任（质量不合格、交期延误）
  □ 知识产权归属（OEM 的 Logo/设计属于你）
```

## 五、头程物流与仓储

### 5.1 物流方案决策

| 方案 | 时效 | 成本 | 适合 |
|------|------|------|------|
| **工厂 → 货代 → 海运 → FBA** | 25~35 天 | 低 | 大批量常规补货 |
| **工厂 → 货代 → 空运 → FBA** | 7~12 天 | 中 | 紧急补货/新品首发 |
| **工厂 → 货代 → 快船 → FBA** | 15~20 天 | 中低 | 主力补货方式 |
| **工厂 → 海外仓 → FBA** | 灵活 | 中 | 需中转/换标/一件代发 |

### 5.2 货代选择

```
筛选要点：
  □ 专注亚马逊头程（不是做普货的）
  □ 有自己的报关团队
  □ 能提供多家船公司选择
  □ 价格透明（无隐性费用）
  □ 能实时追踪物流状态
  □ 出了问题有赔付机制

建议：同时对接 2~3 家货代，分散风险
```

### 5.3 FBA 包装要求

```
外箱要求：
  □ 六面硬质瓦楞纸箱（建议 BC 楞以上）
  □ 箱体完整，无破损，无变形
  □ 不用打包带、松紧带（会被拒收）
  □ 不用泡沫颗粒（会污染仓库）
  □ 外箱贴两张 FBA 箱唛（相邻两面）
  □ 超重标注：>15kg 贴 "Team Lift"，>23kg 贴 "Mech Lift"

产品包装：
  □ 每个产品有独立包装
  □ FNSKU 条码贴在产品包装上（非外箱）
  □ FNSKU 不被折叠/遮挡
  □ 易碎品：气泡膜独立包裹 + 通过 3 英尺跌落测试
```

## 六、供应链成本控制

### 6.1 成本构成全景

```
产品成本 = 采购价 + 包装费 + 头程运费 + 质检费

降本路径：

采购价  → 批量大了重新议价 / 换低一档材质（不降品质）
包装费  → 优化包装设计，减重减体积
头程费  → 拼柜/慢船 / 选择离港口近的供应商
质检费  → 初期全检，品质稳定后降为抽检
```

### 6.2 1688 采购的隐性成本

| 隐性成本 | 说明 | 如何避免 |
|----------|------|----------|
| 沟通成本 | 来回确认耗费时间 | 一次给清所有要求，做确认单 |
| 品质成本 | 大货和样品不一致 | 签合同锁样品、中间品控抽检 |
| 交期成本 | 延误导致断货 | 比约定交期提前 7 天下单 |
| 尾货成本 | 订单多余库存 | 首单少订，后续动态补货 |
| 汇率成本 | 人民币升值压缩利润 | 适当用汇率锁定工具 |

## 七、供应链风险与应对

| 风险 | 应对策略 |
|------|----------|
| **供应商跑路/倒闭** | 不做 100% 预付，保持 2~3 家备选供应商 |
| **原材料涨价** | 签合同时锁定价格有效期 |
| **质检发现批量瑕疵** | 合同约定返工/退货/罚款条款 |
| **春节/节假日停产** | 提前 45 天备货，避开春节前后发货 |
| **环保/限电停产** | 关注供应商所在地政策，提前备安全库存 |
| **物流丢件/损坏** | 买货代保险，跟货代明确赔付条款 |

## 八、供应链升级路线

```
阶段 1：1688 拿货（纯贸易）
  → 找到可靠供应商，稳定品质和交期

阶段 2：OEM 贴牌
  → 有自己的品牌和包装，产品基础款定制

阶段 3：ODM 微创新
  → 基于市场反馈改进产品，工厂配合开发

阶段 4：自有工厂/合资
  → 量大或壁垒高时，掌控生产端
```

> 大多数百万美金级卖家在阶段 2~3，不要急于跳阶段 4。

## 十、供应链 Checklist

### 新供应商引入
- [ ] 营业执照验证（天眼查）
- [ ] 至少拿 2 批样品（间隔 2 周以上）
- [ ] 视频验厂或实地考察
- [ ] 完成供应商评估表（8 维度打分）
- [ ] 签合同（含质量标准+违约责任）
- [ ] 首批小单试跑（200~500 件）

### 每单出货
- [ ] 生产中期抽检（如有必要）
- [ ] 出货前全检/抽样
- [ ] 核对 FBA 装箱要求
- [ ] 拍照留底（每箱内容和外观）
- [ ] 确认物流单号并追踪
