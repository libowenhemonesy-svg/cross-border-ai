# 亚马逊广告ACOS优化实战

> 广告是放大器——好产品被放大成爆款，差产品被放大成亏损。ACOS 是衡量放大效率的核心指标。

---

## 一、广告基础概念

### 1.1 核心公式

```
ACOS = 广告花费 ÷ 广告销售额 × 100%

例：花了 $10 广告费，产生 $50 销售额 → ACOS = 20%
```

### 1.2 ACOS 的三种解读

| 视角 | 含义 | 目标 |
|------|------|------|
| **盈亏视角** | ACOS = 广告成本占售价的比例 | ACOS < 利润率 → 广告本身盈利 |
| **投入视角** | ACOS 是获客成本 | 新品期可接受高 ACOS |
| **权重视角** | 广告出单 → 提升自然排名 → 自然订单增加 | 长期 ACOS 降低 |

### 1.3 TACOS（总广告销售成本）

```
TACOS = 广告花费 ÷ (广告销售额 + 自然销售额) × 100%
```

> 🔥 **ACOS 容易被误读**。广告带动自然排名后，自然订单增加了，但广告订单看起来 ACOS 还是高。**高手看 TACOS，新手看 ACOS。**

---

## 二、广告类型与适用场景

### 2.1 SP 商品推广（Sponsored Products）

| 投放方式 | 说明 | 适用阶段 |
|----------|------|----------|
| **自动广告** | 亚马逊自动匹配关键词/ASIN | 新品期：跑词、跑ASIN数据 |
| **手动-广泛匹配** | 你指定词，亚马逊匹配相关变体 | 拓词期：从自动广告提炼出的好词 |
| **手动-词组匹配** | 搜索词包含指定词组即可触发 | 精准期：核心词组卡位 |
| **手动-精准匹配** | 只有完全相同的搜索才触发 | 收割期：高转化词锁定 |
| **商品投放** | 投放指定ASIN/类目 | 竞品截流、互补品关联 |

### 2.2 SB 品牌推广（Sponsored Brands）

- 搜索顶部品牌展示位
- 需要品牌备案
- 适合做品牌认知、新品系列推广

### 2.3 SD 展示型推广（Sponsored Display）

- 站内外广告位
- 再营销（浏览过但没买的人）
- 竞品防御（投在自己产品页，防止竞品广告抢流量）

## 三、广告结构设计

### 3.1 广告架构金字塔

```
         ┌──────────────┐
         │  品牌广告 1个  │  ← 品牌曝光，占总预算 10%
         ├──────────────┤
         │  手动精准 3~5个│  ← 高转化词收割，占总预算 40%
         ├──────────────┤
         │  手动广泛 1~2个│  ← 拓词，占总预算 25%
         ├──────────────┤
         │  自动广告 1个  │  ← 数据采集，占总预算 25%
         └──────────────┘
```

### 3.2 单广告组结构建议

```
每个广告组内：
  - 关键词数量：10~20 个
  - 关键词类型保持一致（不要广泛和精准混在一个组）
  - 每个组日预算 ≥ $10（低于 $10 数据样本太小）
```

## 四、竞价策略选择

### 4.1 三种竞价模式

| 竞价模式 | 原理 | 适用场景 |
|----------|------|----------|
| **固定竞价** | 每个点击固定出价 | 新品期，需要稳定曝光 |
| **动态-仅降低** | 转化概率低时自动降价 | 稳定期，控制 ACOS |
| **动态-提高与降低** | 转化概率高时加价 50%，低时降价 | 冲排名/大促期间 |

### 4.2 出价公式

```
建议出价 = 目标 ACOS × 产品售价 × 预估转化率

例：目标 ACOS=25%，售价 $30，转化率 10%
建议出价 = 25% × $30 × 10% = $0.75
```

> 💡 实际出价从建议出价的 80% 起步，每 3 天根据表现调整。

## 五、ACOS 优化核心策略

### 5.1 降低 ACOS 的六个杠杆

| 杠杆 | 操作 | 效果 |
|------|------|------|
| **否定投放** | 添加不相关/不转化的搜索词 | ⭐⭐⭐⭐⭐ |
| **降低出价** | 对高 ACOS 词降低 bid | ⭐⭐⭐⭐ |
| **提高转化率** | 优化 Listing 图片/价格/评论 | ⭐⭐⭐⭐ |
| **精准匹配升级** | 广泛→词组→精准，逐步收窄 | ⭐⭐⭐ |
| **暂停低效词** | ACOS > 目标 2 倍的词暂停 | ⭐⭐⭐ |
| **分时调价** | 非高峰时段降低出价 | ⭐⭐ |

### 5.2 否定投放的黄金法则

```
必须否定的词：
  ❌ 完全不相关的搜索词（如卖水杯，被搜 "water bottle stickers"）
  ❌ 点击 > 15 次、0 转化的词
  ❌ 竞品 ASIN 搜索（如果你不是在做商品投放）
  
慎重否定的词：
  ⚠️ 高相关的长尾词但暂时没转化 → 可能有潜力，留着观察
  ⚠️ CPC 高但转化也高的词 → 看整体 ACOS 而非单价
```

### 5.3 关键词的"养"与"杀"

```
关键词生命周期管理：

新增（观察 7 天，消耗 $10~15 预算）
  ↓
  有转化  →  进入「培育组」，加预算，尝试提高出价
  无转化  →  检查相关性 → 不相关立即否；相关再给 7 天
  ↓
培育（转化稳定的词）
  ↓
  ACOS < 目标  →  「明星词」，提高出价抢更多流量
  ACOS > 目标  →  降低出价或转为精准匹配
  ↓
明星（长期高转化的词）
  → 锁定精准匹配，成为你的利润引擎
```

## 六、新品广告打法（0~30天）📖 详见[[亚马逊新品广告预算分配指南|新品广告预算分配指南]]

### 6.1 第 1 周：自动广告启动

```
广告设置：
  类型：SP 自动广告
  日预算：$10 ~ $20
  竞价策略：固定竞价
  默认出价：$0.40 ~ $0.80（视类目而定）

目标：
  → 不是盈利，是收集数据
  → 让亚马逊系统了解你的产品，建立初始关联
```

### 6.2 第 2 周：手动广告加入

```
从第 1 周的自动广告报告中提取：
  - 有转化的搜索词 → 开手动精准
  - 有点击无转化的长尾词 → 开手动广泛/词组

手动广告设置：
  类型：SP 手动
  日预算：$10 ~ $15
  竞价策略：固定竞价
  关键词：10~15 个 C 组长尾词
```

### 6.3 第 3~4 周：拓展优化

```
行动清单：
  □ 自动广告报告分析，否定低效词
  □ 手动广告加 B 组拓展词
  □ 表现好的精准词提高出价
  □ 考虑开启商品投放（投互补品类）
  □ 总日预算提升到 $30 ~ $50
```

## 七、稳定期广告优化（30天+）

### 7.1 广告优化节奏

```
每周必做：
  1. 下载 Search Term Report
  2. 按花费排序，找 Top 10 高花费词
  3. 高花费但低转化的 → 否定或降价
  4. 找到新的高转化词 → 加精准匹配
  5. 调整预算分配
```

### 7.2 广告预算日/周分布

```
非旺季：
  周一到周四：正常预算
  周五到周日：+20% 预算（购物高峰）

旺季（Prime Day / 黑五等）：
  全周期：预算翻倍，竞价策略改成「提高与降低」
```

## 八、常见广告问题诊断

| 症状 | 可能原因 | 解决办法 |
|------|----------|----------|
| 曝光低 | 出价太低、关键词太小众 | 提价 or 换大一点的词 |
| 曝光高、点击低 | 主图差、价格高、标题不行 | 优化 Listing，对比竞品 |
| 点击高、转化低 | 页面承接不住流量 | 检查 A+、评论、价格 |
| ACOS 突然飙升 | 竞品降价、新品入局 | 检查竞品动态，可能需要降 bid |
| 广告花费集中在一个词 | 这个词 bid 太高 | 降价该词，把预算分给其他词 |
| 广告不出单 | 自动广告没跑对类目 | 检查分类节点是否正确 |

## 九、ACOS 目标设定指南

| 阶段 | 合理 ACOS | 策略 |
|------|-----------|------|
| 新品冷启动（0~14天） | 40%~80% | 允许亏损，重在出单和数据 |
| 爬升期（14~60天） | 25%~40% | 逐步优化，追求打平 |
| 稳定期（60天+） | 15%~25% | 广告需盈利 |
| 成熟期（6个月+） | 10%~20% | 精准收割，自然流量为主 |

> ⚠️ 不要用统一 ACOS 目标要求所有产品。高单价高利润产品可以接受更高 ACOS，低单价低利润产品必须严格控制。

- [ ] 分析 ACOS/TACOS 趋势
- [ ] 重新评估关键词分级（A/B/C 组）

- [ ] 调整总预算分配
