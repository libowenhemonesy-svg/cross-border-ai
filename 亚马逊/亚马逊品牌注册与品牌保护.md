# 亚马逊品牌注册与品牌保护

> 商标 = 你在亚马逊上的房产证。没有品牌备案，你只是亚马逊的「租户」，随时可能被赶走。

---

## 一、为什么要做品牌注册

### 1.1 不做品牌的三大风险

| 风险 | 说明 |
|------|------|
| **跟卖无保护** | 你的 Listing 被别人跟卖，你没有任何办法 |
| **无法使用品牌工具** | 没有 A+ 页面、品牌旗舰店、SB 广告、Vine |
| **投诉无门** | 被竞品恶意投诉时，没有品牌做护身符 |

### 1.2 品牌备案带来的权益

```
✅ A+ 页面（EBC）         → 转化率提升 5%~10%
✅ 品牌旗舰店（Store）    → 免费的品牌落地页
✅ SB 品牌推广广告        → 搜索顶部曝光位
✅ 品牌分析（ABA）        → 查看关键词排名、竞品数据
✅ Vine 评论计划          → 合规获取早期评论
✅ 透明计划（Transparency）→ 防止假货跟卖
✅ 零计划（Project Zero） → 自助删除假冒商品
✅ 虚拟捆绑（Virtual Bundle）→ 创建产品组合
```

## 二、品牌注册前置准备

### 2.1 商标类型选择

| 类型 | 说明 | 适合谁 |
|------|------|--------|
| **文字商标** | 注册品牌名称（如 "ANKER"） | 大多数卖家 |
| **图形商标** | 注册 Logo 图案 | 有独特 Logo 的卖家 |
| **组合商标** | 文字 + 图形 | 全方位保护（推荐） |

### 2.2 商标注册地区

| 市场 | 商标注册地 | 建议方式 |
|------|-----------|----------|
| 美国站 | USPTO（美国专利商标局） | 美国商标，最优先 |
| 欧洲站 | EUIPO（欧盟知识产权局） | 欧盟商标，覆盖 27 国 |
| 日本站 | JPO（日本特许厅） | 日本商标 |
| 英国站 | UKIPO（脱欧后独立） | 英国商标 |

### 2.3 注册时间线

```
美国商标注册（USPTO）：
  提交申请 → 3~4 个月 → 审查 → 公告期 30 天 → 注册完成
  全程：约 8~12 个月
  费用：$250 ~ $350/类（官费）+ 代理费 $300 ~ $800

亚马逊品牌备案：
  需商标注册完成（或至少拿到受理回执）
  备案审核：2~15 个工作日
```

> ⚠️ **重要**：亚马逊品牌备案需要商标已经注册（Registered）或至少拿到受理号。TM 标（申请中）现在也支持备案，但功能受限。

## 三、品牌备案实操流程

### 3.1 备案所需材料

```
□ 已注册的商标（或 TM 标受理回执）
□ 商标编号
□ 带有商标的产品图片（实物、非 PS）
□ 包装图片（带有商标的包装）
□ 品牌官网（如有，加分项）
□ 亚马逊卖家账号
```

### 3.2 备案步骤

```
第 1 步：登录 Amazon Brand Registry
  → https://brandregistry.amazon.com/

第 2 步：点击「注册新品牌」

第 3 步：填写品牌信息
  - 品牌名称（必须与商标完全一致）
  - 商标编号
  - 商标注册局（USPTO / EUIPO 等）
  - 商标类型（文字/图形）

第 4 步：填写商品信息
  - 选择有该品牌的产品类目
  - 填写品牌官网/社媒链接（选填）

第 5 步：上传验证图片
  - 产品图：产品上清晰可见品牌标识
  - 包装图：包装上清晰可见品牌标识

第 6 步：提交，等待审核
  - 审核期间会收到一封验证码邮件
  - 将验证码回复给亚马逊（通过 Case）
  - 审核通过后，品牌备案完成 ✅
```

### 3.3 审核不通过的常见原因

| 原因 | 解决办法 |
|------|----------|
| 图片上的商标不清晰 | 重新拍摄高清图 |
| 图片有 PS 痕迹 | 必须实物拍摄，不能后期加 Logo |
| 商标信息与 USPTO 不一致 | 检查大小写、空格是否完全匹配 |
| 商标类别与销售产品不符 | 确认商标选类正确 |

## 四、品牌保护工具详解

### 4.1 Transparency（透明计划）

```
原理：每个产品贴唯一的二维码，亚马逊扫码验真

作用：
  ✅ 完全阻止假货跟卖（跟卖者没有码，无法入库）
  ✅ 消费者扫码验证真伪
  ✅ 扫码可导流到你的品牌页/社媒

费用：约 $0.05/码
适合：高价值易被仿冒的产品
```

### 4.2 Project Zero（零计划）

```
原理：品牌所有者可以直接删除假冒 Listing，无需亚马逊审核

条件：
  - 完成品牌备案
  - 过去 6 个月提交过侵权报告且通过率 > 90%
  
警告：这个权力很大，滥用会被取消资格
```

### 4.3 Brand Gating（品牌门槛）

```
申请品牌门槛后，跟卖者需要：
  ✅ 提供品牌授权书
  ✅ 支付 $1,500 左右的费用
  → 大幅降低跟卖概率

申请条件：完成品牌备案 + 有持续的跟卖侵权记录
```

## 五、品牌侵权投诉与应对

### 5.1 投诉跟卖者

```
投诉路径：
  Brand Registry 后台 → 举报违规 → 选择类型
  → "有人在跟卖我的产品" → 输入跟卖 ASIN → 提交

有效投诉的证据：
  ✅ Test Buy（买一个跟卖的产品，证明是假货）
  ✅ Test Buy 的照片与你的真品对比
  ✅ 你的商标证书
```

### 5.2 商标/版权投诉

```
投诉平台：
  → https://www.amazon.com/report/infringement
  或通过 Brand Registry 后台提交

投诉类型：
  - 商标侵权：对方用了你的品牌名
  - 版权侵权：对方用了你的图片/描述
  - 专利侵权：对方模仿了你的外观设计
  - 假货投诉：对方卖的不是真品
```

### 5.3 如果你被投诉了

```
冷静应对：

第 1 步：分析投诉内容，判断是否真的侵权
第 2 步：如果确实侵权 → 立即下架，联系投诉方和解
第 3 步：如果不侵权 → 准备申诉材料（采购发票、品牌授权等）
第 4 步：写 POA（Plan of Action）提交申诉

⚠️ 不要反复提交无效申诉，每次失败都会降低恢复概率
```

## 六、品牌建设的长期价值

### 6.1 品牌化四个阶段

```
阶段 1：白牌（无商标）
  → 纯卖货，随时可能被跟卖干掉

阶段 2：有标（有商标注册，无品牌意识）
  → 有了基础保护，但还是卖货思维

阶段 3：品牌（有品牌定位，有粉丝）
  → 消费者搜索你的品牌名，而非关键词

阶段 4：品牌矩阵（多品牌/多产品线）
  → Anker 模式：母品牌 + 子品牌覆盖不同品类
```

### 6.2 品牌旗舰店的搭建

```
品牌旗舰店（Amazon Store）是免费的：

必备模块：
  ✅ 品牌故事/理念（Hero 大图）
  ✅ 产品分类展示
  ✅ 热销产品推荐
  ✅ 视频展示区
  ✅ 产品对比表

创意建议：
  → 每季度更新一次
  → 配合促销活动改版（Prime Day / 黑五主题）
  → 用 SB 广告引流到旗舰店
```

### 6.3 品牌分析的利用

品牌备案后才能用的 **ABA (Amazon Brand Analytics)**：

```
可以看到：
  ✅ 热门搜索词的搜索频次排名
  ✅ 竞品 ASIN 的点击和购买占比
  ✅ 你自己产品的复购率
  ✅ 用户画像数据

→ 这些都是选品和广告优化的决策依据
```

## 八、品牌建设 Checklist

### 注册前
- [ ] 确定品牌名称（检查 USPTO 是否已被注册）
- [ ] 确定商标注册类别（与你的产品对应）
- [ ] 选择商标代理或自行申请
- [ ] 准备品牌 Logo（简洁、易识别）

### 注册中
- [ ] 商标提交申请
- [ ] 准备产品/包装实物图（带商标）
- [ ] 确保产品图与商标信息一致

### 备案后
- [ ] 创建 A+ 页面
- [ ] 搭建品牌旗舰店
- [ ] 开启 SB 品牌广告
- [ ] 申请 Vine 评论计划
- [ ] 注册 Transparency（如需）
- [ ] 持续监控跟卖情况
