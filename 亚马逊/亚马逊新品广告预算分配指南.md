# 亚马逊新品广告预算分配指南

> 新品期广告的钱，一半用来买流量，一半用来买数据。只有拿到数据，才知道下一分钱该往哪花。

---

## 一、新品广告的总预算逻辑

### 1.1 预算公式

```
新品期总广告预算 = 目标出单量 × CPC ÷ 预估转化率

拆解：
  目标出单量  → 你想在新品期出多少单（至少 50~100 单才有数据量）
  CPC         → 类目平均点击成本（$0.50 ~ $2.00）
  转化率       → 新品期预估 5%~10%（低于类目平均）
```

### 1.2 预算分级（按卖家类型）

| 卖家类型 | 总预算 | 日预算起点 | 适合 |
|----------|--------|-----------|------|
| **轻量测款** | $300 ~ $500 | $10 ~ $15/天 | 首单测款，不确定产品行不行 |
| **标准推广** | $800 ~ $1,500 | $20 ~ $30/天 | 有一定信心，常规新品打法 |
| **激进打爆** | $2,000 ~ $5,000 | $50 ~ $80/天 | 高价高利润、确定要做爆品 |

### 1.3 预算怎么算出来的

```
例：售价 $29.99，类目平均 CPC $1.20，新品转化率 8%

目标：新品前 30 天出 100 单（每天 3~4 单）
  所需点击 = 100 ÷ 8% = 1,250 次点击
  所需花费 = 1,250 × $1.20 = $1,500

→ 月预算 $1,500，日预算起步 $30~$40，边跑边调
```

## 二、四周预算分配节奏

### 2.1 整体分配比例

```
第 1 周：30%  ← 自动广告为主，跑数据
第 2 周：25%  ← 手动广告加入，开始拓词
第 3 周：25%  ← 优化淘汰，预算向好词集中
第 4 周：20%  ← 精准收割，留预算给下月
```

### 2.2 第 1 周：数据收集期（$300 ~ $450）

```
广告结构：
┌─────────────────────────────────────┐
│ SP 自动广告 × 1                     │
│   日预算：$15 ~ $20                 │
│   竞价策略：固定竞价                 │
│   默认出价：$0.50 ~ $0.80           │
│   作用：跑词、跑ASIN、采集搜索数据   │
└─────────────────────────────────────┘

第 1 周目标：
  ✅ 至少产生 500+ 次展示
  ✅ 至少 15~20 次点击
  ✅ 至少 1~3 个转化（有就行，不求多）
  ✅ 拿到第一批搜索词数据
```

### 2.3 第 2 周：手动词加入（$250 ~ $380）

```
广告结构：
┌─────────────────────────────────────┐
│ SP 自动广告 × 1                     │
│   日预算：$10 ~ $15（适当降低）      │
│   作用：继续跑新词                   │
├─────────────────────────────────────┤
│ SP 手动-精准匹配 × 2~3 个           │
│   日预算：$5 ~ $10 x 2~3 组         │
│   关键词：第 1 周跑出的有转化长尾词   │
│   竞价策略：固定竞价                 │
│   出价：建议出价的 80% 起步          │
│   作用：锁定有转化的精准词            │
└─────────────────────────────────────┘

第 2 周目标：
  ✅ 手动广告产生点击和转化
  ✅ 验证第 1 周的关键词是否可持续出单
  ✅ 开始否定不相关的搜索词
```

### 2.4 第 3 周：优化淘汰期（$250 ~ $380）

```
广告结构：
┌─────────────────────────────────────┐
│ SP 自动广告（保留）                  │
│   日预算：$8 ~ $10                   │
├─────────────────────────────────────┤
│ SP 手动-精准匹配（保留表现好的）     │
│   日预算：好词加钱，差词暂停          │
├─────────────────────────────────────┤
│ SP 手动-广泛匹配 × 1（新增）         │
│   日预算：$8 ~ $12                  │
│   关键词：第 1~2 周跑出的中等表现词   │
│   作用：用广泛匹配拓更多长尾          │
└─────────────────────────────────────┘

第 3 周行动：
  ✅ 高 ACOS（> 目标 2 倍）的搜索词 → 否定
  ✅ 高点击 0 转化的词 → 暂停观察或否定
  ✅ 转化好的词 → 加精准匹配，提预算
  ✅ 表现中等的词 → 转移到广泛匹配组
```

### 2.5 第 4 周：精准收割期（$200 ~ $300）

```
广告结构：
┌─────────────────────────────────────┐
│ SP 自动广告                          │
│   日预算：$5 ~ $8（收敛）            │
├─────────────────────────────────────┤
│ SP 手动-精准匹配 × 3~5 个            │
│   日预算：$15 ~ $25 合计             │
│   锁定高转化词，集中火力              │
├─────────────────────────────────────┤
│ SP 商品投放 × 1（可选）              │
│   日预算：$5 ~ $10                  │
│   投在互补品或弱竞品 ASIN            │
└─────────────────────────────────────┘

第 4 周目标：
  ✅ 广告 ACOS 降到 30%~40%
  ✅ 自然订单开始出现
  ✅ 明确哪些词是主力出单词
  ✅ 为第 2 个月做预算规划
```

## 三、广告类型的预算切分

### 3.1 新品四件套预算占比

| 广告类型 | 预算占比 | 开启时间 | 说明 |
|----------|:---:|----------|------|
| **SP 自动广告** | 35%~40% | 第 1 天 | 必须开，数据之源 |
| **SP 精准匹配** | 30%~35% | 第 2 周 | 最好的精准词 |
| **SP 广泛/词组** | 15%~20% | 第 3 周 | 拓词用 |
| **SP 商品投放** | 5%~10% | 第 3~4 周 | 截流互补品 |
| **SB 品牌广告** | 0% | 先不开 | 新品没评论，品牌广告转化率低 |

### 3.2 为什么新品不打品牌广告

```
SB 广告的新品困境：
  → SB 展示在搜索顶部，曝光很大
  → 但新品评分低（甚至 0 评论）
  → 用户点进去 → 看到 0 评论 → 不买
  → 结果：花了钱、点了、不转化 → ACOS 极高

建议：
  评论数 > 30 + 评分 > 4.2 之后，再开启品牌广告
```

## 四、按产品价格的分级预算

### 4.1 低价产品（<$20）

```
特点：利润薄、广告占比敏感、靠走量
总预算：$300 ~ $600
日预算：$10 ~ $20/天
建议 ACOS：20%~25%

预算分配：
  自动广告：50%（跑大量词，低价广撒网）
  精准匹配：30%
  广泛匹配：20%

⚠️ 低价品广告容错率极低，CPC 超过 $1.0 的类目慎打广告
```

### 4.2 中等价格（$20 ~ $50）

```
特点：有利润空间、广告策略灵活、主流区间
总预算：$800 ~ $1,500
日预算：$25 ~ $40/天
建议 ACOS：25%~35%

预算分配：
  自动广告：35%
  精准匹配：35%
  广泛匹配：20%
  商品投放：10%
```

### 4.3 高价产品（>$50）

```
特点：利润高、决策成本高、转化率低、广告需耐心
总预算：$2,000 ~ $5,000
日预算：$50 ~ $100/天
建议 ACOS：30%~50%（前期可放宽）

预算分配：
  自动广告：30%
  精准匹配：40%  ← 重点在精准词，转化意愿最强的流量
  广泛匹配：15%
  商品投放：15%

⚠️ 高价品转化周期长，不要因为第一周没出单就关广告
```

## 五、日预算的精细化调控

### 5.1 按出单时段调整

```
观察广告报告中的出单时间分布：

如果集中在某个时段 → 预算向该时段倾斜
  （亚马逊目前不支持分时调价，但可以手动调整）

比如：
  出单高峰在美国时间 18:00~22:00（中国时间 6:00~10:00）
  → 在这个时间段前 1 小时稍微提高出价
  → 非高峰时间保持低价
```

### 5.2 预算消耗监控

```
日预算花完的三种情况：

1. 花得恰到好处（预算用掉 90%~100%）
  → 广告正常运转，不需调整

2. 花得太快（上午就花完了）
  → 出价设太高 → 降低出价
  → 或者关键词都是大词 CPC 高 → 换一部分长尾词

3. 花不完（一天下来只花了 30%）
  → 出价太低 → 提价
  → 或者关键词搜索量太小 → 换更大的词
```

### 5.3 预算日波动管理

```
周一~周四：正常预算（100%）
周五~周日：+10%~20%（购物高峰）
Prime Day/黑五等：翻倍

不要做的：
  ❌ 周一把一周的预算全打光
  ❌ 周末不管，让广告超预算断流
```

## 六、广告预算要避的六个坑

| 坑 | 问题 | 正确做法 |
|----|------|----------|
| **一上来就打大词** | CPC $2+空转没转化 | 新品先打长尾词，等权重上来再碰大词 |
| **自动广告出价太高** | 跑出一堆无关词浪费钱 | 从建议出价的 60%~80% 起步 |
| **频繁调预算** | 系统不稳定，数据断断续续 | 至少跑 3~5 天再调整 |
| **否定太快** | 刚点 3 次没转化就否了 | 至少 15 次点击 + 无转化再否定 |
| **没钱了硬撑** | 预算过低，一天就几个点击 | 要么加钱要么聚焦最精准的词 |
| **不设预算上限** | 某天突然烧了 $200 | 每个广告活动必须设日预算上限 |

## 七、一个完整的预算计算表

### 7.1 实战案例：售价 $34.99 的瑜伽垫

```
输入变量：
  售价：$34.99
  目标月出单：120 单（新品期）
  类目平均 CPC：$0.95
  预估新品转化率：7%
  类目平均 ACOS：30%

计算过程：

所需点击 = 120 ÷ 7% = 1,714 次
所需花费 = 1,714 × $0.95 = $1,628

按 ACOS 反算：
  广告带来收入 = 120 × $34.99 = $4,199
  预算 ÷ 收入 = $1,628 ÷ $4,199 = 38.8% ACOS ✅

结论：月预算 $1,600，日预算起步 $40~$50
```

### 7.2 四周预算执行表

| 周次 | 日预算 | 自动 | 精准 | 广泛 | 商品 | 目标 |
|------|:---:|:---:|:---:|:---:|:---:|------|
| W1 | $45 | $30 | $15* | — | — | 跑出数据 |
| W2 | $50 | $20 | $20 | $10* | — | 词开始出单 |
| W3 | $55 | $15 | $25 | $10 | $5 | ACOS 开始降 |
| W4 | $50 | $10 | $25 | $10 | $5 | 锁定主力词 |

> \* 第 1 周的「精准」和第 2 周的「广泛」视第一周数据情况开启

## 八、什么时候该加预算

### 加预算的信号

```
✅ 加预算！：
  → ACOS < 目标值 且 曝光和点击在增长
  → 精准匹配的转化词，ACOS 健康但日预算不够花
  → 类目旺季来临（Prime Day / 黑五 / 圣诞）
  → 竞品出问题（断货/差评），窗口期抢流量

❌ 不加预算：
  → ACOS > 目标值
  → 广告花费在涨但转化没跟上
  → 只是曝光大了，转化率一直很低
```

### 预算追加的幅度

```
正确的加预算节奏：
  每次 +20%~30%，跑 3~5 天观察效果
  效果好 → 继续加
  效果不好 → 退回原来预算

错误的做法：
  今天 $20，明天直接拉到 $100（系统重新学习，波动巨大）
```

## 九、预算到第二个月的过渡

```
第 1 个月结束时的状态：
  → 哪些词能出单，ACOS 多少，已经清楚
  → Listing 开始有评论（至少 5~10 条）
  → 自然订单开始出现

第 2 个月的预算分配：
  总预算不变（或微调），但结构变化：

  自动广告：25%（收敛，不再大量拓词）
  精准匹配：40%（集中火力在验证过的词）
  广泛匹配：20%（拓新词）
  商品投放：15%（扩大截流范围）
  品牌广告：0%~5%（评论 > 20 后开启）

第 2 个月目标：
  ACOS 降到 25%~30%，TACOS 降到 20% 以下
```

## 十、预算管理 Checklist

### 产品上架前
- [ ] 计算目标出单量 → 反算总预算
- [ ] 查类目平均 CPC，确认预算是否够用
- [ ] 确定四周的日预算和广告结构
- [ ] 设置每个广告活动的日预算上限
- [ ] 准备关键词清单（A/B/C 三组）

### 第 1~2 周
- [ ] 每天检查预算是否正常消耗
- [ ] 第 3~4 天提取自动广告的搜索词
- [ ] 第 5~7 天开启手动精准
- [ ] 否定明显不相关的搜索词

### 第 3~4 周
- [ ] 按 ACOS 对关键词分组（留/观察/否）
- [ ] 高转化词提高出价和预算
- [ ] 准备第 2 个月的预算方案
- [ ] 分析自然订单占比
