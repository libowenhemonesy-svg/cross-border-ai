# 亚马逊关键词调研方法

> 关键词 = 流量入口。在亚马逊，你的 Listing 能排到哪些词的前面，决定你能吃到多少自然流量。

---

## 一、关键词的基础认知

### 1.1 关键词的三个层级

```
一级（核心大词）  →  搜索量大、竞争激烈、转化率低
二级（精准词）    →  搜索量中等、意图明确、转化率高
三级（长尾词）    →  搜索量小、竞争低、转化率极高
```

**示例**：卖 "不锈钢保温杯"

| 层级 | 关键词 | 搜索量 | 竞争 | 转化率 |
|------|--------|--------|------|--------|
| 一级 | water bottle | 200K+ | 极高 | 低 |
| 二级 | stainless steel water bottle | 30K | 高 | 中 |
| 三级 | insulated stainless steel water bottle 24oz for gym | 2K | 低 | 极高 |

> 💡 **核心策略**：新品期先打三级词，站稳后扩展到二级，最后吃一级词的长尾流量。

---

## 二、关键词的四大来源

### 2.1 亚马逊搜索下拉框

```
在搜索框输入你的产品核心词 → 亚马逊会自动推荐
↓
这些推荐词 = 真实用户的搜索习惯 = 高价值关键词
```

**实操**：
```
搜 "water bottle" → 下拉出现：
  water bottle with straw         ← 长尾词
  water bottle for kids           ← 人群词
  water bottle stainless steel    ← 属性词
  ...
```

> 🔥 每个下拉词都按字母逐个遍历：`water bottle a`, `water bottle b`, `water bottle c`...

### 2.2 竞品 ASIN 反查

用 Helium 10 **Cerebro**，输入 Top 10 竞品 ASIN：

```
Cerebro 输出：所有竞品有排名的关键词列表
↓
筛选条件：
  - 搜索量 > 500
  - 竞品排名在 Top 20 的词
↓
得到 → 「这个类目最重要的关键词清单」
```

### 2.3 搜索词报告（Search Term Report）

从广告后台下载「搜索词报告」：

```
顾客搜索词 = 你的广告实际被触发的搜索词
↓
分析：
  - 哪些词带来了点击？（加到手动广告）
  - 哪些词被点击但没有转化？（否定投放）
  - 哪些词的低竞品ASIN点击？（说明你的Listing在这个词下弱，需要强化）
```

### 2.4 跨平台关键词

| 平台 | 作用 | 工具 |
|------|------|------|
| Google Trends | 看搜索趋势，判断季节性 | 免费 |
| Pinterest | 发现场景词、使用方式的表述 | 免费 |
| YouTube | 竞品视频标题中的关键词 | 免费 |
| Reddit/论坛 | 用户怎么「说」这个产品 | 免费 |

> 🔍 关键词调研不只是工具活，更是 **对用户语言的理解**。用户在 TikTok 上怎么叫你的产品，搜索框里就怎么搜。

## 四、关键词筛选与分级

### 4.1 关键词打分矩阵

对每个词从 4 个维度打分（1~5 分）：

| 关键词 | 搜索量 | 相关性 | 竞争度 | 转化潜力 | 总分 | 优先级 |
|--------|--------|--------|--------|----------|------|--------|
| water bottle | 5 | 3 | 1 | 1 | 10 | ⭐⭐ |
| stainless steel water bottle | 4 | 5 | 3 | 4 | 16 | ⭐⭐⭐⭐ |
| insulated gym water bottle | 3 | 5 | 4 | 5 | 17 | ⭐⭐⭐⭐⭐ |

> 总分越高，越优先打。

### 4.2 关键词分组

将筛选后的词按意图分为 3 组：

```
【A 组 — 首打词】（5~10 个）
  条件：高相关性 + 搜索量 1000~5000 + 竞争度低~中
  用途：新品期手动广告核心词、标题核心词

【B 组 — 拓展词】（10~20 个）
  条件：高相关性 + 搜索量 500~2000
  用途：广泛匹配广告、五点描述嵌入

【C 组 — 长尾词】（不限）
  条件：所有能找到的长尾变体
  用途：Search Terms 后端关键词、自动广告拓词
```

## 五、关键词在 Listing 中的嵌入策略

### 5.1 嵌入优先级

```
标题（权重最高） > 五点描述 > A+ 内容 > 后端 Search Terms
```

### 5.2 标题关键词公式

```
[品牌] + [A组核心词1] + [A组核心词2] + [属性词] + [场景词] + [规格]
```

**关键规则**：
- ✅ 前 80 个字符放最重要的关键词
- ✅ 自然读起来通顺
- ❌ 不要堆砌（keyword stuffing）
- ❌ 不要用逗号分隔一堆词

### 5.3 后端 Search Terms

```
规则：
  - 用空格分隔，不要用逗号
  - 不用重复 Listing 中已出现的词
  - 不用品牌名、ASIN、临时性描述（"new"）
  - 不用主观词（"best", "amazing"）
  - 总长度不超过 250 字节

正确示例：insulated gym water bottle stainless steel 24oz bpa free leak proof
错误示例：best water bottle, amazing quality, cheap price
```

## 六、关键词排名追踪

### 6.1 追踪工具

| 工具 | 特点 |
|------|------|
| Helium 10 Keyword Tracker | 追踪任意 ASIN 在任意词下的排名变化 |
| Jungle Scout Rank Tracker | 类似功能 |
| Seller Sprite（卖家精灵）| 中文界面 |

### 6.2 追踪频率

```
新品期（前 1~2 个月）：每周追踪
稳定期：每两周追踪
成熟期：每月追踪
```

### 6.3 看排名变化该做什么

| 排名变化 | 可能原因 | 行动 |
|----------|----------|------|
| 📈 稳定上升 | Listing 权重在积累 | 保持策略，逐步加预算 |
| 📉 突然下降 | 竞品降价/新品入局 | 检查竞品动态，考虑是否降价/加广告 |
| ➡️ 长期不动 | 触达天花板 | 开新词、拓长尾、做变体 |
| 🎢 大起大落 | 不稳定，权重不够 | 补评论、优化转化率 |

## 七、关键词策略进阶

### 7.1 关键词矩阵打法

```
核心大词 (1~2个) → 广泛匹配 → 目标：跑出长尾词数据
精准词 (3~5个)   → 词组匹配 → 目标：稳定出单
长尾词 (10+个)   → 精准匹配 → 目标：高转化低ACOS
竞品ASIN词       → 商品投放 → 目标：抢对手流量
```

### 7.2 季节词打法

如果产品有季节性波动（如圣诞礼品、夏季游泳用品）：

```
旺季前 1.5 月：开始打季节相关关键词
旺季中：预算拉满，所有季节词全开
旺季结束前 2 周：逐步降价清库存，关季节词广告

📊 用 Google Trends 提前看搜索趋势的启动时间点
```

### 7.3 新品关键词启动策略

```
第 1 周：只打 C 组长尾词（精准匹配）
         → 低 CPC，高转化率，快速积累销售历史
第 2 周：加入 B 组拓展词（词组匹配）
         → 拓展流量来源
第 3 周：加入 A 组首打词（广泛匹配）
         → 此时 Listing 有一定权重，才能在大词下竞争
```

> ⚠️ **新手常犯错误**：一上来就打大词。Listing 没有权重时打大词 = 花广告费帮竞品引流。

## 八、关键词调研完整流程

```
第 1 步：定位核心产品词
  → 你的产品「最本质」叫什么？（water bottle / desk lamp / yoga mat）

第 2 步：收集所有相关关键词
  → 下拉框 + Cerebro + Magnet + 竞品标题 → 导出 Excel

第 3 步：数据清洗
  → 去重、过滤无关词、标注搜索量/竞争度

第 4 步：打分分级
  → 用打分矩阵分出 A/B/C 三组

第 5 步：关键词嵌入
  → A 组进标题，B 组进五点/A+，C 组进 Search Terms

第 6 步：广告匹配
  → A 组手动广告，B 组广泛匹配，C 组自动广告拓词

第 7 步：持续迭代
  → 每周看 Search Term Report，加高转化词，否低转化词
```

---

## 九、关键词调研 Checklist

- [ ] 至少收集 50+ 个相关关键词
- [ ] 用 Cerebro 反查 Top 10 竞品的关键词
- [ ] 完成关键词打分矩阵
- [ ] 分出 A/B/C 三组
- [ ] A 组关键词全部嵌入标题
- [ ] B 组关键词分布在五点描述和 A+ 中
- [ ] C 组和未嵌入的词放入后端 Search Terms
- [ ] 建立了关键词排名追踪表
- [ ] 制定了第 1~4 周的广告关键词策略
