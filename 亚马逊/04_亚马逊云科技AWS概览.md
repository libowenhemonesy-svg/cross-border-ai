# 亚马逊云科技（AWS）业务概览

## 服务简介

亚马逊云科技（Amazon Web Services，简称 AWS）是全球最大的云计算平台，提供超过 200 项全功能服务，涵盖计算、存储、数据库、机器学习、人工智能、物联网、安全等领域。

---

## 核心服务分类

### 1. 计算服务
| 服务 | 说明 |
|------|------|
| Amazon EC2 | 弹性虚拟云服务器，可伸缩的计算容量 |
| AWS Lambda | 无服务器计算，按代码执行时间付费 |
| Amazon ECS / EKS | 容器编排服务（Docker / Kubernetes） |
| AWS Fargate | 容器的无服务器计算引擎 |
| AWS Batch | 批量计算作业处理 |

### 2. 存储服务
| 服务 | 说明 |
|------|------|
| Amazon S3 | 对象存储，可存储任意数量数据 |
| Amazon EBS | 块存储卷，用于 EC2 实例 |
| Amazon EFS | 弹性文件系统 |
| Amazon Glacier | 低成本归档存储 |
| AWS Storage Gateway | 混合云存储集成 |

### 3. 数据库服务
| 服务 | 说明 |
|------|------|
| Amazon RDS | 关系数据库服务（MySQL/PostgreSQL/Oracle/SQL Server） |
| Amazon DynamoDB | 全托管 NoSQL 键值和文档数据库 |
| Amazon Aurora | 与 MySQL/PostgreSQL 兼容的关系数据库 |
| Amazon ElastiCache | 内存缓存服务（Redis/Memcached） |
| Amazon Redshift | PB 级数据仓库服务 |
| Amazon Neptune | 全托管图数据库 |

### 4. 网络与内容分发
| 服务 | 说明 |
|------|------|
| Amazon VPC | 虚拟私有云，逻辑隔离的网络环境 |
| Amazon CloudFront | 全球内容分发网络（CDN） |
| Amazon Route 53 | 可扩展的 DNS 域名系统 |
| AWS Direct Connect | 专用网络连接到 AWS |
| Elastic Load Balancing | 应用负载均衡 |

### 5. AI 与机器学习
| 服务 | 说明 |
|------|------|
| Amazon SageMaker | 构建、训练和部署机器学习模型 |
| Amazon Bedrock | 构建和扩展生成式 AI 应用 |
| Amazon Q | AI 助手（企业版） |
| Amazon Rekognition | 图像和视频分析 |
| Amazon Polly | 文本转语音服务 |
| Amazon Lex | 构建对话式 AI 聊天机器人 |
| Amazon CodeWhisperer | AI 编程助手 |

### 6. 分析服务
| 服务 | 说明 |
|------|------|
| Amazon EMR | 大数据处理框架（Hadoop/Spark） |
| Amazon Kinesis | 实时数据流处理 |
| Amazon QuickSight | 商业智能和可视化 |
| AWS Glue | 无服务器数据集成 |
| Amazon Athena | 交互式查询（SQL on S3） |

### 7. 安全与身份管理
| 服务 | 说明 |
|------|------|
| AWS IAM | 身份和访问管理 |
| AWS Shield | DDoS 防护 |
| AWS WAF | Web 应用防火墙 |
| Amazon GuardDuty | 威胁检测服务 |
| AWS KMS | 密钥管理服务 |

## 客户群体

AWS 服务覆盖各行各业：
- **互联网/科技**：Netflix、Airbnb、Uber、Spotify
- **金融**：Capital One、纳斯达克
- **医疗**：辉瑞、强生
- **制造**：西门子、宝马
- **政府**：美国政府机构、NASA
- **中国客户**：小米、OPPO、虎牙等

## 中国站点信息

- **中国官网**：aws.amazon.cn
- **运营模式**：由光环新网（北京区域）和西云数据（宁夏区域）运营
- **服务特点**：与全球 AWS 服务隔离，符合中国法规要求
